# Bilge'nin Yapay Zekâ Atölyesi  
## AI'HAN ACADEMY

**Bilge'nin Yapay Zekâ Atölyesi**, çocuklara Yapay Zekâ okuryazarlığını oyunlaştırılmış, etkileşimli ve güvenli bir öğrenme deneyimiyle kazandırmak için geliştirilmiş web tabanlı bir eğitim portalıdır.

Portal; çocukların Yapay Zekâ kavramlarını yalnızca teorik olarak öğrenmesini değil, deneyerek, oynayarak, sorgulayarak ve güvenli kullanım ilkelerini uygulayarak keşfetmesini amaçlar.

Bu proje **AI'HAN ACADEMY** çatısı altında, çocuk dostu dijital öğrenme deneyimleri geliştirme yaklaşımıyla hazırlanmıştır.

# Bilge's Artificial Intelligence Workshop  
## AI'HAN ACADEMY

**Bilge's Artificial Intelligence Workshop** is a browser-based interactive educational portal designed to help children understand Artificial Intelligence through playful, safe, and hands-on learning experiences.

The portal helps children explore AI concepts by experimenting, playing, questioning, creating, and practicing safe technology use.

This project was developed under **AI'HAN ACADEMY** as part of a child-friendly digital learning approach.

---

## 🇬🇧 English

## 🎯 Purpose

This workshop is designed to help children understand Artificial Intelligence in a responsible, safe, ethical, and creative way.

Main goals:

- Explain Artificial Intelligence in a child-friendly way
- Show why data is important for AI systems
- Gamify algorithms, model training, and decision-making processes
- Let children experience AI through vision, sound, drawing, storytelling, music, and coding activities
- Teach that AI-generated answers must always be checked
- Build awareness of privacy and personal data protection
- Encourage children to become creative, critical, and ethical technology users

---

## 🤖 Who Is Bilge?

**Bilge** is a friendly robot guide who supports children throughout the Artificial Intelligence journey.

Bilge:

- Gives helpful hints
- Guides children through workshop missions
- Provides safety reminders
- Reminds learners to check AI-generated answers
- Motivates children to complete tasks

---

## 🧩 Portal Modules

The portal includes several AI learning areas:

### 🧠 Foundations of Artificial Intelligence
What is AI, how does it work, and how is it different from human intelligence?

### 🔎 Data Workshop
Data types, examples, missing data, and incorrect data.

### 🧭 Algorithm Workshop
Understanding step-by-step problem solving.

### ⚙️ Model Training
Learning how a system can make predictions from examples.

### ⚖️ Decision Making
Understanding why AI answers must be checked.

### 👁️ Computer Vision
Exploring how AI can detect objects from camera input or visual examples.

### 🎙️ Voice and Speech
Text-to-speech, listening, speaking, and speech-to-text experiences.

### 🎨 Generative AI
Prompt writing, visual idea generation, and creative production.

### 🎵 Music AI
Rhythm, repetition, tempo, and emotion in music generation.

### 📖 Story Workshop
Creating stories with characters, settings, problems, and solutions.

### 💻 Coding Workshop
Giving step-by-step commands to move a robot.

### 🛡️ Safe AI Use
Learning not to share private information, checking answers, and using AI kindly.

### 🎮 AI Games
Mini games that reinforce AI concepts.

### 🏅 Certificate Hall
Creating a personalized certificate after completing the workshop tasks.

---

## 🏆 Certificate System

Children who complete the workshop missions can receive **Bilge's Artificial Intelligence Workshop Certificate**.

The certificate system includes:

- Participant name field

## 💻 Local Usage

To run the project locally:

1. Download the project file.
2. Rename the main file as `index.html`.
3. Open `index.html` in a modern browser.
4. The portal will run directly.

No additional installation is required.

---

## 🌐 Publishing with GitHub Pages

To publish the project with GitHub Pages:

1. Create a new GitHub repository.
2. Upload the project file.
3. Rename the main HTML file as `index.html`.
4. Go to the repository **Settings**.
5. Open the **Pages** section.
6. Select the `main` branch and `/root` folder.
7. GitHub Pages will generate a public link for the portal.
